# Wiscoursealert

## Setup

1. Set up environment file `.env` in `backend` directory

```python
PORT = 3000
MONGODB_URI =
REDIS_HOST =
REDIS_PORT =
REDIS_PASS =

MAIL_USERNAME =
MAIL_PASSWORD =
OAUTH_CLIENTID =
OAUTH_CLIENT_SECRET =
OAUTH_REFRESH_TOKEN =
URL_MAIN =
```

## Run

1. Run back-end

```shell
cd backend
npm start
```
